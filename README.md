Stopwatch Web App

A simple and responsive stopwatch web application built using HTML, CSS, and JavaScript. It allows users to start, stop, reset, and resume time tracking in a clean and minimal user interface.


---

Features :

Start the stopwatch

Pause/Stop the stopwatch

Resume from paused state

Reset the stopwatch to 00:00:00

Responsive and clean design


---

Technologies Used :

HTML5 – For structuring the webpage

CSS3 – For styling the stopwatch UI

JavaScript – To handle stopwatch logic (timing, events)
